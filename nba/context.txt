NBA ESPN API Instructions

Link for teams: https://site.api.espn.com/apis/site/v2/sports/basketball/nba/teams
sports.leagues.teams = List of all 30 teams
sports.leagues.teams.team = Team container
sports.leagues.teams.team.id = Team id
sports.leagues.teams.team.abbreviation = Team abbreviation
sports.leagues.teams.team.displayName = Team full name (ex. Atlanta Hawks)
sports.leagues.teams.team.shortDisplayName = Team short name (ex. Hawks)
sports.leagues.teams.team.color = Team color code (ex. c8102e)
sports.leagues.teams.team.alternateColor = Team alternate color code (ex. fdb927)

sports.leagues.teams.team.logos.href = Team logo link

sports.leagues.teams.team.logos.rel = Team logo specification (ex. "full", "default")


Link for scoreboard/games: https://site.api.espn.com/apis/site/v2/sports/basketball/nba/scoreboard?dates={YYYYMMDD}
events.id = Game id
events.competitions.competitors.id = Team id
events.competitions.competitors.homeAway = Will say if team is home or away ("home" or "away")
events.competitions.competitors.team.id = Team id
events.competitions.competitors.team.displayName = Team full name
events.competitions.competitors.team.shortDisplayName = Team short name
events.competitions.competitors.score = Team score
events.competitions.competitors.record = Team record

^ all are the same for the second team. Each team is wrapped inside a container in competitors

events.competitions.notes.headline = Game headline (ex. "East 1st Round - Game 6")
events.competitions.status.displayClock = Current game clock
events.competitions.period = Current period
events.competitions.type.description = Game status (Scheduled, Final)


Link for boxscore, play-by-play and linescore: https://cdn.espn.com/core/nba/boxscore?xhr=1&gameId={gameId}
gamepackageJSON.boxscore.players = Boxscore for players on both teams
gamepackageJSON.boxscore.players.displayOrder = Order of team (1 or 2)
gamepackageJSON.boxscore.players.team.shortDisplayName = Team short name
gamepackageJSON.boxscore.players.color = Team color
gamepackageJSON.boxscore.players.id = Team id
gamepackageJSON.boxscore.players.abbreviation = Team abbreviation
gamepackageJSON.boxscore.players.statistics.names = List of all stat names (short form)
gamepackageJSON.boxscore.players.statistics.athletes.starter = Indicates if player is a starter (true or false)
gamepackageJSON.boxscore.players.statistics.athletes.athlete.shortName = Athlete short name (ex. "L. James")
gamepackageJSON.boxscore.players.statistics.athletes.athlete.id = Athlete id
gamepackageJSON.boxscore.players.statistics.athletes.athlete.position.abbreviation = Athlete position abbreviation
gamepackageJSON.boxscore.players.statistics.athletes.stats = Player stats. Goes in order according to stat names
gamepackageJSON.boxscore.players.statistics.athletes.didNotPlay = Indicates if player played (true or false) if true, set all stats for player to 0
gamepackageJSON.boxscore.players.statistics.athletes.active = Indicates if plyer is on the court (true or false)

gamepackageJSON.plays.sequenceNumber = Current play sequence number
gamepackageJSON.plays.period.displayValue = Current period (ex. "1st Quarter")
gamepackageJSON.plays.homeScore = Home score
gamepackageJSON.plays.clock.displayValue = Current time on clock
gamepackageJSON.plays.team.id = Team id of current play
gamepackageJSON.plays.awayScore = Away score
gamepackageJSON.plays.text = Play text
gamepackageJSON.plays.participants.athlete.id = Participating athlete id

gamepackageJSON.header.competitions.competitors.homeAway = Will say if team is home or away ("home" or "away")
gamepackageJSON.header.competitions.competitors.score = Team score
gamepackageJSON.header.competitions.competitors.winner = Winner or loser (true or false)
gamepackageJSON.header.competitions.competitors.id = Team id
gamepackageJSON.header.competitions.competitors.team.abbreviation = Team abbreviation
gamepackageJSON.header.competitions.competitors.team.linescores.displayValue = Linescore value for each quarter. In their own separate parameters
